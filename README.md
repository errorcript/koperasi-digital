# Koperasi Digital Ecosystem (Kop-Warung)

A modern solution to bridge cooperatives and local shops (warungs).

## 🚀 Demo Routes
- `/` - **Member Landing Page & Member App Mockup** (Interactive Cost Calculator included)
- `/dashboard` - **Admin Dashboard** (Management & Transaction Logs)
- `/mitra` - **Warung Mitra App Mockup** (Sales History & Commission Monitoring)

## 🛠 Tech Stack
- **Next.js 14** (App Router)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## 📦 Local Setup
```bash
npm install
npm run dev
```

## 🌐 Deployment
To deploy on Vercel:
```bash
npx vercel deploy --prod
```

## 💰 Cost Estimation (MVP)
- **Initial Dev**: ~Rp 25.000.000
- **Server**: ~Rp 450.000/month
- **Payment Gateway**: 0.7% (QRIS)
